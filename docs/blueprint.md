# **App Name**: VPAC Client Care

## Core Features:

- Google Meet and MS Teams Integration: Enable seamless video call scheduling and launching via Google Meet and Microsoft Teams, including OAuth and API access in settings.
- Availability Checker: Scans staff calendars (Google & Microsoft) to ascertain next available time slots for scheduling via API access.
- Appointment Slot Definition: Staff define available appointment blocks using an intuitive drag interface, covering the upcoming month.
- Client Appointment Booking: Clients select any available 15-minute increment within the defined time slots.
- Confirmation and Reminders via Quo: Appointment confirmations and day-of reminders sent via Quo (OpenPhone) API upon client time slot requests.
- Branded Shortlink Generation: Generate a branded shortlink to the online meeting within the appointment reminders.

## Style Guidelines:

- Background color: Almost white (#FAFAFA), creating a crisp, clean backdrop.
- Primary color: Soft blue (#72BCD4) for a professional, calming feel.
- Accent color: Muted orange (#BCAAA4), to indicate actions.
- Font: 'Inter', a grotesque-style sans-serif for its modern and neutral appearance, used for both headings and body text.
- Use simple, professional icons relevant to scheduling and video conferencing.
- Employ a clean, white layout that is crisp and airy.
- Incorporate subtle animations during calendar interactions.
